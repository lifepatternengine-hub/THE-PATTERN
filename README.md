# THE PATTERN

A diagnostic platform for creative people in midlife transition.

## Live site
https://life-pattern-engine.xyz

## Structure
- `index.html` — main page
- `articles.template.html` — source template for the articles index
- `articles.html` — generated by the build (gitignored)
- `scripts/build-articles.mjs` — pulls Status=Published rows from Notion and renders `articles.html`
- `api/subscribe.js`, `api/confirm.js` — newsletter double-opt-in
- `api/rebuild.js` — cron handler that triggers a fresh deploy

## Articles pipeline
1. Add a row to the Notion `The Pattern — Article Pipeline` database. Required fields for the site: Title, Slug, Pattern (Friction/Dissolution/Reconstruction), Excerpt, Cover (URL), Read time, Published (date).
2. Flip `Status` to `Posted`. Cron runs Tuesdays 08:00 UTC and triggers a redeploy.
3. The build runs `scripts/build-articles.mjs` which queries Notion and stamps the cards into `articles.html`.

### Required env vars (Vercel)
- `NOTION_TOKEN` — Notion integration secret (already in use for newsletter)
- `NOTION_ARTICLES_DB_ID` — `6c425d1840164cbe84951bf6482d5959`
- `VERCEL_DEPLOY_HOOK_URL` — Deploy Hook URL from Vercel project settings → Git
- `CRON_SECRET` — optional shared secret for `/api/rebuild`

## Deploy
Connected to Vercel — push to `main` branch auto-deploys.

## Development
- Homepage: open `index.html` directly (no build step).
- Articles index: run `node scripts/build-articles.mjs` once, then open `articles.html`.
